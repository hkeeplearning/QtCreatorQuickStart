## Qt中的事件

一个对象表示一个事件，事件不同于信号，比如：单击一下界面上的鼠标按钮，会产生鼠标事件QMouseEvent，不是鼠标产生的；因为按钮被按下，所以他会发射clicked单击信号（按钮产生的）。

事件与信号是两个不同层面的东西。发出者不同，作用也不同。
